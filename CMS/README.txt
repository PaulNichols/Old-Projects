How to view this sample:

1) Create a new asp.net solution.
2) Add a reference to the aspmenu.dll as specified in the user manual (http://aspmenu.net/Support/UserManual.aspx).
3) Add the following files to the project:
	aspmenuexample.aspx
	ASPMenuMain.ascx
	aspmenu.css
	aspmenu.xml
	all the files in the images folder.
4) Run the solution.
