WinDataGrid_Customers
   How to use the Windows DataGrid
Overview
This class is based on a System.Windows.Forms.DataGrid class and was built to display the Customers table content.

To do so, this class calls the spS_Customers stored procedure. You can load the whole content of the table or only a subset of the records.

Basically, you only have to call the two following methods to use this class:
  • Initialize
  • RefreshData

This DataGrid is completely editable. The four basic operations are supported:
  • Insert
  • Update
  • Delete
  • Select
Once the user has made his changes, it can be committed by simply calling the CommitChanges() method.
