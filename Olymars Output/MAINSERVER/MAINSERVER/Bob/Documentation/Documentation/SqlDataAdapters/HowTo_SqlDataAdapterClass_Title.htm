SqlDataAdapter_Title
   How to use the SqlDataAdapter class
Overview
This class allows you to build a complete and functional SqlDataAdapter for the Title table. It is located in the Bob.SqlDataAdapters namespace.

The four basic operations (Insert, Update, Delete and Select) are supported.

It relies on the following stored procedures:
  • spI_Title
  • spU_Title
  • spD_Title
  • spS_Title
Once initialized, you can use the FillDataSet method to fill a DataSet which can, for example, be used with a DataGrid.
