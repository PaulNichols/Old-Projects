Installation
In order to use successfully this Web form, you have to perform the following tasks:
  • Create a virtual directory (Bob_WebForms in our example) pointing on a given directory (WebSiteDir in our example)
  • Create a Bin sub directory in WebSiteDir and copy Bob.Forms.dll in it
  • Copy the WebFormList_Title.aspx file in the WebSiteDir directory
  • Copy the web.config file in the WebSiteDir directory
  • Edit the web.config file and update the Bob ConnectionString key to reflect the correct location of the SQL Server database.
  • You can now browse to http://YourWebServer/Bob_WebForms/WebFormList_Title.aspx
Reusing this Web Form
