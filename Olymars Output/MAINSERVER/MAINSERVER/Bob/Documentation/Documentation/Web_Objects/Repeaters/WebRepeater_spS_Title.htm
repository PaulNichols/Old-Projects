WebRepeater_Title
   How to use the Web Repeater
Overview
This class is based on a System.Web.UI.WebControls.Repeater class and was built to display the Title table content.

To do so, this class calls the spS_Title stored procedure. You can load the whole content of the table or only a subset of the records.

Basically, you only have to call the two following methods to use this class:
  • Initialize
  • RefreshData
