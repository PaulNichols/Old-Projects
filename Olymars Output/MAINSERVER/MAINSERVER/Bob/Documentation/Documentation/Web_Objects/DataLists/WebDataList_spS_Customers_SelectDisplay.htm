WebDataListCustom_spS_Customers_SelectDisplay
   How to use the Web DataList
Overview
This class is based on a System.Web.UI.WebControls.DataList class and was built to display the resultset returned back by the spS_Customers_SelectDisplay stored procedure. You can choose which field you want to display and which one is to be considered as a primary key.

You can load the whole content of this resultset or only a subset of the records.

Basically, you only have to call the two following methods to use this class:
  • Initialize
  • RefreshData
