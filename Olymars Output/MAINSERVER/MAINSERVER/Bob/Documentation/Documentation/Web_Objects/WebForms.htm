Bob.Web.Forms namespace
   Overview
Overview
This namespace contains the System.Web.UI.Page based class for each of your database tables that has at least one primary key.

Those classes are "standard" Web Form classes and for each table, two different forms are created:
  • A form that allows you to Add, Update and Delete records from the corresponding table.
  • A form that allows you to View the corresponding table content in a list view mode.
