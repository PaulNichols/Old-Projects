   Overview on how to update an existing record
Overview
In order to update an existing record of this table, a specific stored procedure was created: [spU_Title]. With this stored procedure you can update only one of the table column if this is what is wanted. For each column, you can specify how to treat Null values.

This stored procedure has the following parameters to be supplied before its execution:
  • @TitleId:
    Unique Identifier


  • @Title:
    Title Description


  • @ConsiderNull_Title: This parameter allows you to specify how to behave if a Null value is affected to the @Title parameter.
    • If @Title = Null and @ConsiderNull_Title = False, no change is made to the [Title] column.
    • If @Title = Null and @ConsiderNull_Title = True, Null is affected to the [Title] column.
