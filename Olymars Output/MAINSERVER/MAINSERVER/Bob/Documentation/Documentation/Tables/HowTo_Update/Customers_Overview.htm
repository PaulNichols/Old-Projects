Overview
In order to update an existing record of this table, a specific stored procedure was created: [spU_Customers]. With this stored procedure you can update only one of the table column if this is what is wanted. For each column, you can specify how to treat Null values.

    Customer unique identifier


  • @CompanyName:
    Company for which the contact represents
