Customers
   Overview on how to retrieve some records
Overview
In order to retrieve some records from the Customers table, you can call 4 differents stored procedures depending on your need:

[spS_Customers]: this stored procedure allows you to retrieve:
  • all the table records
  • only one specific record
  • all the table records related to the following foreign tables:
    • Title (via the TitleId column)
This stored procedure allows you to get back the data in a resultset or in an XML stream. This stored procedure returns the following columns:
  • CustomerID: Customer unique identifier
  • CompanyName: Company for which the contact represents
  • ContactName: Customer or Contact Name (if a business)
  • TitleId: Foriegn Key for Contact Title
  • Address: Customer/Contact Address
  • City: Customer/Contact City
  • PostalCode: Customer/Contact Post code
  • Phone: Customer/Contact Phone
  • Email: Email Address
  • WebAddress: Web Address URL
  • Fax: Customer/Contact Fax
  • Active: Active flag
[spS_Customers_Display]: this stored procedure allows you to retrieve:
  • all the table records "display" information
  • only one specific record "display" information
  • all the table records "display" information related to the following foreign tables:
    • Title (via the TitleId column)
This stored procedure returns the following columns:
  • ID1: Customer unique identifier
  • Display: Display for this table.
[spS_Customers_SelectDisplay]: this stored procedure allows you to retrieve:
  • all the table records and all the foreign tables "display" information
  • only one specific record and all the foreign tables "display" information
  • all the table records and all the foreign tables "display" information related to the following foreign tables:
    • Title (via the TitleId column)
This stored procedure allows you to get back the data in a resultset or in an XML stream. This stored procedure returns the following columns:
  • CustomerID: Customer unique identifier
  • CompanyName: Company for which the contact represents
  • ContactName: Customer or Contact Name (if a business)
  • TitleId: Foriegn Key for Contact Title
  • TitleId_Display: Display of the Title table
  • Address: Customer/Contact Address
  • City: Customer/Contact City
  • PostalCode: Customer/Contact Post code
  • Phone: Customer/Contact Phone
  • Email: Email Address
  • WebAddress: Web Address URL
  • Fax: Customer/Contact Fax
  • Active: Active flag
[spS_Customers_Full]: this stored procedure allows you to retrieve:
  • all the table records and all the foreign tables columns
  • only one specific record and all the foreign tables columns
  • all the table records and all the foreign tables columns related to the following foreign tables:
    • Title (via the TitleId column)
This stored procedure allows you to get back the data in a resultset or in an XML stream. This stored procedure returns the following columns:
  • CustomerID: Customer unique identifier
  • CompanyName: Company for which the contact represents
  • ContactName: Customer or Contact Name (if a business)
  • TitleId: Foriegn Key for Contact Title
  • Address: Customer/Contact Address
  • City: Customer/Contact City
  • PostalCode: Customer/Contact Post code
  • Phone: Customer/Contact Phone
  • Email: Email Address
  • WebAddress: Web Address URL
  • Fax: Customer/Contact Fax
  • Active: Active flag
  • TitleId_TitleId: Unique Identifier
  • TitleId_Title: Title Description
