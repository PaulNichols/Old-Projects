OlymarsDemo.SqlDataAdapters namespace
   Overview
Overview
This namespace stores all the SqlDataAdapters classes. Each of these classes represents a fully ready-to-use SqlDataAdapters. You have one SqlDataAdapter per table. Each of this SqlDataAdapter can automatically handle the four basic operations you can perform on a table:
  • Insert
  • Update
  • Delete
  • Select
You can use this class in conjunction with a DataGrid.
