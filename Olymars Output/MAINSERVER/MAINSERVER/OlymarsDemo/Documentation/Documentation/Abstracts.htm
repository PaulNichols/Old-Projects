Overview
The OlymarsDemo.AbstractClasses namespace provides "abstract" classes that allow you to retrieve records information from tables very easily.

For each table of the OlymarsDemo database that has at least one primary key, an abstract class is provided.

This class is synchronized with the table schema in the sense that it provides properties that exactly map to the table columns for a very easy value retrieving.
