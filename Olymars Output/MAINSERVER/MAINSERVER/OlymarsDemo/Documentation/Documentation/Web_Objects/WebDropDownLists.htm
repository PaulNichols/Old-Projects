OlymarsDemo.Web.DropDownLists namespace

Overview
This namespace contains the System.Web.UI.WebControls.DropDownList based class for each of your database tables that has one and only one primary key and for each stored procedure known to retrieve one or more resultsets.

Those classes are "standard" Web DropDownLists classes that populate themselves with the corresponding table content or the resultset returned by the stored procedure. Only two lines of code are required to use them.
