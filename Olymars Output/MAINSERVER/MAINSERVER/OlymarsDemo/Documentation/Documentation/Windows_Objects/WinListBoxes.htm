OlymarsDemo.Windows.ListBoxes namespace
   Overview
Overview
This namespace contains the System.Windows.Forms.ListBox based class for each of your database tables that has one and only one primary key and for each stored procedure known to retrieve one or more resultsets.

Those classes are "standard" Windows ListBoxes classes that populate themselves with the corresponding table content or the resultset returned by the stored procedure. Only two lines of code are required to use them.
