Overview
This namespace contains the System.Windows.Forms.DataGrid based class for each of your database tables and for each stored procedure known to retrieve one or more resultsets.

Those classes are "standard" Windows DataGrids classes that populate themselves with the corresponding table content. Only two lines of code are required to use them.
